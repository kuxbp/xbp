■Git Bashのダウンロード

GitBashのインストールファイルをここからダウンロードします。

■インストール

ダウンロードしたGitBashのインストールファイルをダブルクリックしてインストールすます。インストール途中の設定は下記を参考にしてください。

最初の画面はそのままNext


次の画面は赤枠の部分にチェックを付け足す


その次の画面はそのままNext


その次の画面もそのままNext


次の画面は赤枠を選択


その次の画面はそのままNext


次の画面は赤枠を選択


最後の画面は全てチェックを外す（どっちでもいいけど）
